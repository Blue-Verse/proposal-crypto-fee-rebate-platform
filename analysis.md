## 2. URL/이미지 분석

### 벤치마크 1: 테더맥스 (tethermax.io)

- **사이트 유형**: 가상화폐 거래소 수수료 페이백 비교 플랫폼
- **기술 스택**: Next.js App Router, SSR 기반
- **핵심 기능**:
  - 거래소별 수수료율 및 페이백 비율 비교 테이블
  - API 직접 연동 자동 정산 시스템
  - 실시간 수수료 계산기
  - 사용자 대시보드 (환급 현황, 정산 내역)
  - SEO 최적화된 거래소별 상세 페이지
- **연동 거래소**: 4개 (바이낸스, 바이비트, OKX, 비트겟)
- **UI 특징**: 깔끔한 카드 기반 비교 UI, 반응형 지원
- **차별점**: API 자동 정산으로 수동 정산 대비 높은 정확도

### 벤치마크 2: 곰지네페이백 (gomzipayback.com)

- **사이트 유형**: 거래소 수수료 페이백 비교/중개 플랫폼
- **기술 스택**: SPA (React 기반 추정)
- **핵심 기능**:
  - 거래소별 혜택 카드형 비교 페이지
  - 레퍼럴 가입 유도 랜딩 페이지
  - 공지사항/이벤트 관리
  - 카카오톡 상담 연동
- **연동 거래소**: 9개 (바이낸스, 바이비트, OKX, 비트겟, MEXC, Gate.io 등)
- **UI 특징**: 카드 기반 시각적 비교, 모바일 최적화
- **차별점**: 다수 거래소 지원, 간결한 정보 전달

### 분석 종합

두 벤치마크 모두 **거래소 비교 + 레퍼럴 페이백** 모델. 테더맥스는 API 자동 정산에 집중, 곰지네페이백은 거래소 커버리지에 집중. 클라이언트 프로젝트는 MVP이므로 초기 2개 거래소(Bitget, OKX) 수동 정산 + 비교 페이지 중심으로 구현, 향후 API 자동 정산 고도화 로드맵 제안.
